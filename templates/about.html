🌍 About EcoTrack AI

EcoTrack AI is an AI-powered Carbon Footprint Management System that helps users calculate, monitor, and reduce their carbon emissions.

The system provides carbon emission calculations, AI recommendations, progress tracking, reports, eco tips, and downloadable PDF reports.

This project is developed using:

  • Python Flask
  • HTML5
  • CSS3
  • Bootstrap 5
  • SQLite Database
  • Chart.js
